# Zgrada+ (Stanar+)

Frontend aplikacija za upravljanje stambenim zgradama: nadzorna ploča, zgrade, suvlasnici, dužnici, radni nalozi, uplatnice i ostalo.

## Tehnologije

- Vite, TypeScript, React
- shadcn/ui, Tailwind CSS

## Pokretanje

```sh
npm install
npm run dev
```

Backend API konfigurira se putem env varijabli (npr. `VITE_API_URL`). Vidi `.env.example` ako postoji.

## Build

```sh
npm run build
```
